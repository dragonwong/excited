2. 他来自扬州，戴着一副黑框眼镜，知识渊博，饱览诗书，去过西方许多国家，曾为国家生死来到北京，靠治我的奋斗，把握历史的行程，常常给人一些人生经验，和朱物华院长有着特别的关系。现在，很多年轻人能够背诵他说过的话。没错，他就是——朱自清。
